Configure Webhook
  1. Log in to your Payu Merchant Dashboard.
  2. Go to Developer → Webhooks .
  3. Toggle the Test Mode switch in the right menu to try the integration without charging customers.
  4. Click on Create Webhook.
  5. Select the type of webhook as Payments and choose the desired event (e.g., Successful, Failed, Refund, or Dispute).
    Note: You can select only one event at a time. To configure multiple events, repeat the process and create a separate webhook for each event.
  6. Under the "Webhook URL", enter your Odoo database URL followed by /payment/payu/webhook in the Webhook URL field.
    For example: https://example.odoo.com/payment/payu/webhook
  7. Enable the following events:
    • Successful
    • Failed

Configuration on Odoo

  1. Install the module from the Apps menu.
  2. Now, navigate to the Payment providers by going to:
    • Accounting ‣ Configuration ‣ Payment Providers
    • Website ‣ Configuration ‣ Payment Providers
    • Sales ‣ Configuration ‣ Payment Providers
  3. Select the provider and configure the Credentials tab.
  4. Set the State to Enabled.

After configuring the details, On your website checkout page, you will see the Payu payment methods (UPI, Card, Netbanking, etc.) based on the configuration you set in the Payment Provider.
